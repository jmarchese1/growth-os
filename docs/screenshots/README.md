# Screenshots

Drop real PNGs here with these exact filenames and they'll render in the main [README](../../README.md) automatically.

## Expected files

| Filename | Shows | Recommended dimensions |
|---|---|---|
| `hero.png` | Big top-of-page hero | 1600×900 |
| `onboarding.png` | Business onboarding flow | 1200×800 |
| `platform.png` | Platform admin dashboard | 1200×800 |
| `voice-agent.png` | Voice agent configuration | 1200×800 |
| `chatbot.png` | Chatbot preview with a sample business | 1200×800 |

## How to capture

1. Run `pnpm dev` and open the admin platform at `http://localhost:3011`
2. Use Chrome's DevTools device emulator at ~1200×800
3. Cmd/Ctrl+Shift+P → "Capture screenshot" for a clean, chrome-less grab
4. Optimize through [tinypng.com](https://tinypng.com) before committing
