## How to capture

1. Run `pnpm dev` and open the admin platform at `http://localhost:3011`
2. Use Chrome's DevTools device emulator at ~1200×800
3. Cmd/Ctrl+Shift+P → "Capture screenshot" for a clean, chrome-less grab
4. Optimize through [tinypng.com](https://tinypng.com) before committing
